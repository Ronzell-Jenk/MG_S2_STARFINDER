## Canon Rules

1. Use the XM30 naming standard consistently.
2. Keep Recon Platoon and Cavalry Platoon distinct by role.
3. Keep structure, tactics, and battle drills in separate files.
4. Use canon pages for assumptions, doctrinal principles, and echelon ownership.
5. Treat scenario overlays as annexes, not baseline doctrine.
