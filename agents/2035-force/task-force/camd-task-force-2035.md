# CAMD Task Force 2035

## Purpose
Define a modular CAMD task-force unit that can maneuver as a combined-arms package and complete assigned tactical tasks inside a brigade/division operation.

## Mission
The CAMD Task Force maneuvers to:
- gain and retain contact with enemy maneuver elements,
- create local overmatch at decisive points,
- secure routes, crossings, and support nodes,
- enable follow-on force exploitation.

## Core Design
- **Task-organized:** Built per mission from standing platoons and enablers.
- **Combined-arms:** Recon, cavalry screen, mechanized assault, and precision support integrated under one commander.
- **Commandable:** One clear command node with delegated control measures and battle handoffs.
- **Sustainable:** Support element sized to keep movement, fires, and ISR continuous.

## Baseline Task-Force Package
- **HQ Element**
  - Task Force Commander
  - Executive Officer
  - Fires/EW/UAS integration cell
- **Recon Package**
  - 1× Recon Platoon (persistent ISR and route development)
  - 1× Cavalry Platoon (security, contact handoff, screen line control)
- **Maneuver Package**
  - 1× Mechanized Platoon (main-effort assault element)
  - 1× Weapons Platoon detachment (anti-armor and precision strike)
- **Support Package**
  - Combat sustainment node (fuel, ammo, recovery, casualty collection)
  - Maintenance/recovery teams forward-positioned by phase line

## Maneuver Framework
### Phase 1 — Shape
- Recon and cavalry establish contact and map threat dispositions.
- EW/UAS cell confirms target signatures and enemy control nodes.
- Weapons detachment sets initial anti-armor deny zones.

### Phase 2 — Fix
- Cavalry fixes enemy forward security elements.
- Mechanized element occupies assault positions under concealment.
- Fires package disrupts enemy reserves and crossing defenses.

### Phase 3 — Assault
- Mechanized platoon executes main effort on selected axis.
- Recon secures flank observation and confirms breach effects.
- Weapons detachment suppresses enemy anti-armor threats.

### Phase 4 — Consolidate and Continue
- Task force secures objective, re-establishes screen lines, and restores ISR.
- Sustainment node pushes resupply forward to preserve tempo.
- Commander re-frames follow-on task (exploit, defend, or handoff).

## Minimum Control Measures
- Named phase lines for each maneuver phase.
- Primary and alternate support-by-fire positions.
- Triggered handoff points between Recon, Cavalry, and Mechanized elements.
- Decision points tied to enemy reserve commitment and bridge/corridor control.

## Success Criteria
The unit has accomplished its task when it:
- holds or denies assigned key terrain/objective areas,
- preserves maneuver freedom for higher headquarters,
- retains enough combat power to continue operations without operational pause.
